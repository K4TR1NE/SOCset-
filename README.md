## Установка и настройка

Следуйте этим инструкциям, чтобы запустить Vega локально:

1.  **Клонирование репозитория:**
    ```bash
    git clone [URL вашего репозитория]
    cd vega
    ```
2.  **Установка зависимостей:**
    ```bash
    npm install  # или yarn install
    ```
3.  **Настройка переменных окружения:**
    *   Создайте файл `.env` на основе `.env.example`.
    *   Заполните необходимые переменные (ключи API, настройки базы данных и т.д.).
4.  **Миграция базы данных (если необходимо):**
    ```bash
    npm run migrate  # или yarn migrate (зависит от вашей системы миграций)
    ```
5.  **Запуск серверов (frontend и backend):**
    ```bash
    npm start  # или yarn start (обычно запускает оба сервера)
    # или
    npm run start:frontend
    npm run start:backend
    ```
